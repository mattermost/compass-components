### Measurement Units

Balance is maintained in layouts by leveraging standardized units of measurement. 8px is the standard unit of
measurement for all layout elements. Smaller elements like icons and text can use 4px as the standard unit.

<img src={measurementImg} alt={'measurement units'} />

### Spacing

Spacing follows the same standard measurements established above. There are 11 standard spacers that follow
this convention.

### Padding

Padding refers to the spacing within a container and gives breathing room to the content within. Padding should follow
the same measurement and spacing standards by utilizing the 8px unit of measurement.

<img src={paddingImg} alt={'padding examples'} />

## Grids

### 8px grid

As much as possible, all elements should align to an 8px grid.

<img src={grid8pxImg} alt={'example of 8px grid'} />

### 4px grid

Icons, type, and other small elements can align to a 4px grid.

<img src={grid4pxImg} alt={'example of 4px grid'} />

### vertical/baseline grid

A baseline grid of 4px is set to establish a vertical rhythm. 8px is the resulting x-height of our base body copy font
size (14px). We’ve chosen to cut that in half to 4px in order to provide enough flexibility for various layout elements.
8px should be the target grid in cases, but when smaller increments are needed, the 4px baseline can be used. With this
in mind, line heights for body copy should be a multiple of 4px.

<img src={baselineImg} alt={'example of the baseline measurement'} />

### horizontal grid

A horizontal grid is defined to help align UI elements and create a clean, organized layout that feels grounded and
trustworthy. This kind of organization also helps users scan content and find what they’re looking for.

<img src={anatomyImg} alt={'example of the horizontal grid anatomy'} />

export const Template = (args) => {
    const { childCount, padding, margin, colPadding, colMargin, ...rest } = args;
    const content = new Array(childCount).fill('gridChild');
    return (
        <Shape borderWidth={1} width={800} height={400} background={'rgb(240, 210, 180)'}>
            <Grid margin={GridSpacing.all(margin)} flex={1}>
                <Shape borderWidth={1} border background={'rgb(210, 240, 180)'}>
                    <Grid flex={1} padding={GridSpacing.all(padding)} {...rest}>
                        {content.map((key, i) => (
                            <Shape key={key + i} borderWidth={1} background={'rgb(240, 210, 180)'}>
                                <Grid margin={GridSpacing.all(colMargin)} flex={1}>
                                    <Shape borderWidth={1} background={'rgb(210, 240, 180)'}>
                                        <Grid flex={1} padding={GridSpacing.all(colPadding)}>
                                            <Shape borderWidth={1} background={'#F3F3F3'} />
                                        </Grid>
                                    </Shape>
                                </Grid>
                            </Shape>
                        ))}
                    </Grid>
                </Shape>
            </Grid>
        </Shape>
    );
};
