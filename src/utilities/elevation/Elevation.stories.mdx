<Subtitle>Elevation is the relative distance between objects along the z-axis.</Subtitle>

Elevation refers to the z-index depth that an element sits on. The base Elevation is 0 and has no depth.
Subsequent layers increment this Elevation value. In total, there are 6 Elevation levels, not including the base level 0.

### Elevations Values

#### Elevation 1

-   Message Hover Actions
-   Message Attachments
-   Cards

#### Elevation 2

-   Message Attachments Hover
-   Cards Hover

#### Elevation 3

-   Tooltip

#### Elevation 4

-   Popover Parent Menu
-   Channel Member Popover
-   Profile Popover
-   Status Popover
-   Emoji Popover
-   Autocomplete Popover Menus

#### Elevation 5

-   Popover Child Menu

#### Elevation 6

-   Modals

### Elevations & Dark Themes

The Elevation shadows don't show up as well on dark themes, so the shadow styles need to be updated
to increase the opacity values by 4x. For example, Elevation 1 normally has a shadow with an opacity of 8%,
but in dark themes, it has an opacity of 32%.
